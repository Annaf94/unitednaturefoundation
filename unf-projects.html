# United Nature Foundation — Website Every page is a self-contained HTML file (CSS/JS built in). There is now one folder, "media", containing the header video and logo -- everything else is flat. ## Files - index.html, our-story.html, canopy-crew.html, unf-projects.html, donate.html, join-now.html - pathways.html, power.html, pedal.html -- project placeholder pages - media/header-video.mp4 -- homepage hero background video (compressed from the original for web use) - media/unf-logo.png -- logo now used in the nav bar on every page ## How to upload to GitHub 1. Go to your repo -> Add file -> Upload files 2. Drag in all the .html files plus README.md at the top level (overwrites old versions) 3. Then create the "media" folder: drag in the "media" folder itself (containing header-video.mp4 and unf-logo.png) -- GitHub does support drag-and-drop of a single folder like this - If that doesn't work, create a new file inside GitHub named "media/unf-logo.png" and "media/header-video.mp4" using the upload dialog, which will auto-create the media folder 4. Commit, then Vercel will auto-redeploy ## What changed in this update - Homepage hero now plays your header video on loop (muted, autoplay) instead of a placeholder - Homepage headline is now in capital letters - The UNF logo image now appears in the navigation bar on every page, replacing the text-based logo
